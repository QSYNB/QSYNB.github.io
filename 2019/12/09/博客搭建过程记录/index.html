博客搭建过程记录（一）

创建时间:2019-12-09 23:59
阅读:

在经历想要实现部署的个人服务器上并实现

在观看了b站教程后手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo，成功实现了把个人博客部署在git上（windows平台）~过程不算复杂，一个小时就可以搞定。为了防止下一次修改忘记，写一篇文章做一下笔记。

安装node.js，git等步骤略去不表。从在git bash上操作hexo开始记录，以下操作均在相同文件夹。

hexo init

初始化博客，用于在空文件夹中生成博客。

hexo n "文章名称"

创建新文章，创建后可在文件夹中打开。我采用了Typora软件进行编辑。（对markdown还不够熟悉，熟悉之后会直接用语言来写）

hexo s

启动博客，启动后可以再本地4000端口进行预览。网址： http://localhost:4000/ 。按Ctrl+c停止预览。

hexo d

向服务器端推送博客。成功后即可通过域名浏览。成功后需要一定时间才能看到更新。

git clone https://…git地址…

下载新主题。更改博客_config.yml文件，theme字段后的文字。主题的index和具体修改方法，在文件夹下的theme文件夹中修改。

hexo clean

清理博客文件夹。

hexo g

生成相关文件。每次修改后要执行这个命令再执行 推送命令。
