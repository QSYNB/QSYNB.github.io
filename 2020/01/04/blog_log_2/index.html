    在hexo中插入图片

    最开始用了markdown语法中自带的图片命令，但是查了相关资料后发现并不好用。

    在hexo官方文档中发现了正确的姿势。

    前期准备：

    1 把主配置文件_config.yml 里设置post_asset_folder: true

    2 安装插件 在hexo目录下执行cnpm install hexo-asset-image –save， 这样new新文章的时候可以生成一个该文章对应的图片资源文件夹。

    3.

    用语句插入图片，但是没有发现如何设置图片在文章中显示的名称和设置图片大小（有看到文章说用css来设置但是没有找到正确的方法，可恶！html明明是大一学的内容。虚度的光阴唉……）。

    mardown中左右分栏
